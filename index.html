公開中のツール一覧

Online Webpage IDE

ブラウザ上で完結するWebpageのための開発環境。学習者向けに、環境構築等不要でHTML, CSS, JSの3つの最小限のファイルを編集可能で、即座に変更をプレビュー可能。

 GitHub Repository ➔

Nexus Search

検索をメインに。簡易的な検索から、詳細な検索まで対応。検索エンジンの設定も可能。ブックマークを登録できる。

文字数カウント

簡易的な文字数カウントとして使用することが可能。推定読了時間や、原稿用紙にすると何枚分か、段落や行数はいくつかなどを計測可能。

ダミーテキストジェネレーター

一部の有名な著作権切れの文章などをベースに、仮の文章として使うための文章を出力。

Project MMO用日本語化リソースパック

Project MMOが(おそらく)最新のバージョンだと日本語化に対応していないため、その日本語化を実施するための非公式的なリソースパック。
